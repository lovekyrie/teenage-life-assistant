# Excel 导入模板说明

上传 `.xlsx` 文件，需包含以下工作表：

## Sheet: 积分行为

| 类型 | 类别 | 行为名称 | 分值 | 说明 | 每日上限 | 排序 |
|------|------|----------|------|------|----------|------|
| add | 生活 | 自己穿衣服 | 2 | | 1 | 1 |
| add | 学习 | 完成练习 | 3 | | | 2 |
| subtract | 行为 | 乱发脾气 | 2 | | 3 | 1 |

- 类型：`add` 或 `subtract`
- 每日上限、排序可留空

## Sheet: 奖励

| 名称 | 所需积分 | 描述 | 库存 |
|------|----------|------|------|
| 看一集动画 | 10 | 15分钟 | |
| 小玩具 | 50 | | 5 |

- 库存留空表示无限

导入为覆盖式：会清空当前家庭的所有行为和奖励后重新写入。
